Live demo

This LIVE demonstration of many JRobin capabilities was provided by Cherry Network Monitoring project. It uses JRobin as a RRD data backend and graphing tool.

The demo monitors various aspects of system utilization (CPU load, FTP and Apache statistics, traffic graphs, number of emails in the message queue etc) by creating real-time JRobin graphs.

Demo opens in a separate window.
