Support

Things you should know:

  • Some important general questions are already answered on our FAQ page.
  • All JRobin related questions should be posted to our Forum site. Sooner or later all questions get answered.
  • JRobin has no mailing list for its end-users.
  • If you find a bug or a serious performance bottleneck, please report it to our bug tracker so that you can monitor bug resolution progress. Please, don't email developers directly.
  • JRobin shares the same concept and logic with RRDTool. If you are a novice RRD users, these RRDTool man pages might be helpful.
