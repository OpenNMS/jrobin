Download JRobin

Primary JRobin tarball can be freely downloaded from SourceForge.net.

The tarball contains all JRobin class libraries, full product documentation (javadoc and all HTML pages from this web site), complete source code, additional resource files, some external libraries and a single (ant) script which can be used to rebuild the whole distribution.

Some demo-applications from the main package use huge data files not included in the main distribution. You can download these data files separately from the same place (just look for a package named jrobin demo data).

You can use JRobin in your projects under terms of LGPL licence.

CVS access

Current JRboin snapshot can be downloaded from the CVS at  SourceForge.net. JRobin's CVS tree has four different modules:

  • src: contains all Java source files
  • res: contains resource files (used only for MRTG-demo application)
  • lib: contains external Java libraries (used only for MRTG-demo application)
  • ant: contains single jakarta ant script file

Instructions how to checkout CVS files from SourceForge can be found here.

If you want to be notifed by e-mail when our CVS gets updated, subscribe to our JRobin-commits mailing list.
